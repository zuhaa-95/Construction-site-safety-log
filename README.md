Construction-site-safety-log

An automated safety monitoring workflow built with n8n that receives hazard reports from construction sites and instantly triggers emergency responses based on severity.


 🎯 What It Does
When a worker submits a hazard report the system automatically:
 Notifies supervisor instantly through multiple channels
 Fetches live weather & air quality data
 Splits into Critical or Routine path based on severity
 Calculates a Risk Score out of 100
 Logs everything to Google Sheets


 🛠️ Technologies Used
 n8n, Gmail, Discord, Google Sheets
 Open-Meteo API (Weather & Air Quality)
 Ntfy (Push Notifications)
 JavaScript (Custom Logic)
 Public Holiday API
